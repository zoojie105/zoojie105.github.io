Ready Set Glow

마케팅 프로모션 기능 개발

작업기간: 2022.02 ~ 2022.06

특정 상품의 판매 촉진을 위한 프로모션 시스템을 개발

기술 개요 및 구현 방식

  • SNS 리뷰 링크 검증
  • JavaScript & jQuery
    • JavaScript와 jQuery를 사용하여 제출된 링크가 유효한지 확인하기 위해 HTTP 상태 코드를 체크하는 로직 구현
  • PHP
    • 프로모션 상품 구매 자격 확인: 고객이 제출한 SNS리뷰 링크가 유효한 경우 재구매 진행

System Process

  • 고객이 특정 상품을 9.99달러에 구매
  • 고객이 SNS에 구매 상품에 대한 리뷰 작성
  • 고객이 리뷰 링크를 시스템에 제출
  • 시스템이 제출된 링크의 상태 코드를 검증하여 유효한 리뷰 링크인지 확인
  • 링크가 유효하다면, 고객은 다시 9.99달러에 상품을 재구매할 수 있는 권한을 얻는다
