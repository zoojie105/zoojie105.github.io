PayPal 간편결제

PayPal 간편결제 및 스마트버튼 연동 개발

작업기간: 2022.07 ~ 2022.10

사용자에게 보다 안전하고 효율적인 결제시스템을 제공하기 위해 페이팔 스마트 버튼 및 간편결제 API를 웹사이트에 연동

페이팔 간편결제 및 스마트버튼 연동을 통해 사용자가 원하는 결제방식을 선택할 수 있도록 구현

기술 개요 및 구현 방식

  • PayPal SDK를 사용해 백엔드 시스템을 개발하고, REST API를 설계하고 구현
  • JavaScript & jQuery
    • 프론트엔드에서 주문 데이터의 검증 및 처리를 위해 JavaScript와 jQuery를 사용
    • 결제 전 주문 데이터의 유효성을 검증하고, 유효하지 않은 경우 사용자에게 적절한 피드백을 제공하여 UX를 향상
    • jQuery를 통해 사용자의 결제 버튼 클릭 이벤트를 핸들링하여 페이팔 결제 요청을 초기화하는 로직 구현
  • PayPal SDK
    • PayPal SDK를 이용하여 결제 요청 및 주문 정보 생성 처리
    • 사용자가 결제를 승인한 후에는 PayPal로부터 결제 승인 응답을 받아, 이를 토대로 자사의 주문 생성 로직 실행
  • PHP
    • 서버 사이드에서는 PHP를 사용하여 주문 데이터를 처리하고 데이터베이스에 저장하는 로직 구현
    • PayPal로부터 결제 승인을 받으면, 이를 바탕으로 자사의 주문을 생성하고, 이후에 PayPal에 결제 승인 요청 전송
    • 최종적으로 결제가 완료되면 이 정보를 사용자에게 전달하고 데이터베이스에 저장
